# Secure IoV Authentication using Blockchain and BFT

## Overview

This project focuses on building a secure authentication system for the Internet of Vehicles (IoV) using blockchain technology. The goal is to remove dependency on centralized systems and ensure safe communication between vehicles.

## Problem

In current IoV systems, authentication is mostly centralized, which creates problems like data tampering, spoofing, and single point of failure. This makes the system less secure and unreliable.

## Solution

To solve this, we designed a decentralized system using blockchain and the BFT (Byzantine Fault Tolerance) algorithm. Each vehicle is given a unique identity, and authentication is verified through the blockchain network instead of a central server.

## Technologies Used

* Python
* Blockchain concepts
* Machine Learning
* TensorFlow
* OpenCV
* SQL

## How it Works

Vehicles send authentication requests, which are processed through a blockchain network. The BFT algorithm ensures that even if some nodes fail or act maliciously, the system still works correctly. All transactions are stored securely in the blockchain.

## Results

The system showed better performance compared to traditional methods:

* Reduced authentication delay
* Improved security
* Better scalability
